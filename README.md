# L100 – Git Practice Repository

## About This Practice
I created this repository to practice Git commands in a hands-on way and to build a clear understanding of how version control works in real projects.

## What I Practiced
During this exercise, I worked on:
- Creating and cloning repositories
- Tracking changes using git status
- Staging files with git add
- Creating commits with meaningful messages
- Viewing history using git log
- Working with branches and merging
- Pushing changes to GitHub

## Key Learnings
Through this practice, I understood how Git tracks changes step by step and how branching helps in managing features separately.  
This exercise improved my confidence in using Git for real development workflows.

## Purpose of This Repository
The goal of this repository is to document my Git learning journey and demonstrate practical understanding rather than just theoretical knowledge.
