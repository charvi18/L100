## Stash, Reset and Reflog Practice

### Git Stash
Git stash temporarily saves uncommitted changes so you can work on something else.

Commands:
git stash
git stash list
git stash apply
git stash pop

Practice:
1. Modify a file
2. Run git stash
3. Switch branch
4. Run git stash pop to restore changes

---

### Git Reset
Git reset moves HEAD to a previous commit.

Types:
git reset --soft HEAD~1   → undo commit, keep changes staged
git reset --mixed HEAD~1  → undo commit, keep changes unstaged
git reset --hard HEAD~1   → undo commit and discard changes

Practice:
1. Make a commit
2. Run git reset --soft HEAD~1
3. Observe commit removed but changes remain

---

### Git Reflog
Git reflog tracks all HEAD movements and helps recover lost commits.

Commands:
git reflog
git checkout <commit-hash>

Practice:
1. Run git reset --hard HEAD~1
2. Run git reflog
3. Restore commit using checkout
